# 🛡️ CyberGuard Pro - AI Phishing Detector

Advanced AI-powered phishing detection system that analyzes messages and URLs to protect users from online scams.

## ✨ Features

- **AI-Powered Detection**: Machine learning model trained on 1000+ samples
- **Real-time Analysis**: Instant results with confidence scores
- **Trusted Domain List**: Recognizes legitimate companies
- **Detailed Explanations**: Shows why a message is flagged
- **Security Center**: Educational resources and emergency contacts
- **Responsive Design**: Works on desktop and mobile
